### Getting Started

1. Fork the repository
2. Clone your fork
```
git clone https://github.com/tauraigombera/whatsup_gpt.git
```

3. Create a new branch
```
git checkout -b feature/your-feature-name
```

4. Install dependencies
```
 pip install -r requirements.txt
```

5. Create a `.env` file
```
OPENAI_API_KEY=your_key_here
```

6. Run locally
```
python main.py
```

### How to Contribute

### Report Bugs
- Open an Issue
- Describe expected vs actual behavior
- Include logs or screenshots if possible

### Suggest Features
- Open an Issue with the label `enhancement`
- Explain the use case and benefit

### Submit Code Changes
1. Make your changes in a separate branch
2. Follow clean and readable code practices
3. Ensure the app runs without errors
4. Submit a Pull Request (PR)
5. Clearly describe what your PR does

### Code Style

- Use Python 3.11+
- Follow PEP8 standards
- Keep functions small and focused
- Add comments where logic is complex

### Security Notice

Do NOT commit:
- `.env`
- API keys
- Credentials and any sensitive information
